# animation_tutorial

This is an exercise in implementing animations in Flutter.

branchの説明
- main --- refactoring/withAnimatedBuilderブランチと同様
- develop --- refactoring/withAnimatedBuilderブランチと同様
- use/AnimatedWidget --- AnimatedWidgetを使用してサイズが変化するアニメーションの実装
- refactoring/withAnimatedBuilder --- AnimatedWidgetブランチをAnimatedBuilderを使ってリファクタリング
- create/SimultaneousAnimations --- 1つのオブジェクトに対して2つのアニメーション（サイズ/不透明度）の実装
